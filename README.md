# Otimiza-o-Moderna-e-Segura-de-Rede-Windows-10-11
Este repositório apresenta um script moderno, seguro e compatível com Windows 10/11 para limpeza e ajuste da pilha de rede. O objetivo não é “hackear” a internet, mas restaurar e ajustar corretamente o que o próprio Windows já faz bem, evitando tweaks obsoletos que ainda circulam na internet.
